
# 📦 Asset Catalog (Eco & Pro)

プライベートグループ（最大5名）向けの、軽量・高機能な素材管理サーバーです。
Macでの制作ワークフローに最適化されており、通信量の節約と隠しファイルの管理に対応しています。

## 🛠 運用・更新ルール

新しい素材を追加した際は、以下の手順でインデックス（`list.json`）を更新してください。

1. **素材の追加**: MacのFinder等で `assets/` フォルダ内にファイルを配置し、GitHubへ `push` します。
2. **スキャン実行**: カタログページを開き、右上メニューの **「APIスキャン」** をクリックします。
3. **JSONの保存**: スキャン完了後、表示された **「JSON保存」** ボタンを押し、`list.json` をダウンロードします。
4. **リポジトリへ反映**: ダウンロードした `list.json` を `assets/` フォルダ直下に上書きし、GitHubへ `push` してください。

## 💡 特徴と便利な機能

### 1. 通信容量の節約（Eco Mode）

- **1MBを超えるファイル**: 初期状態ではプレビューを読み込みません。カード上の「表示」ボタンを押した時だけデータを取得します。
- **一括表示**: ヘッダーの「全メディア表示」ボタンで、すべてのプレビューを強制的に読み込めます。

### 2. 隠しファイル・フォルダの管理

- フォルダ名が `.` で始まるもの（例: `.secret/`）は、**「🙈 HIDDEN」** セクションとしてリストの最下部に隔離されます。
- これらも初期状態ではプレビューが非表示になります。

### 3. 表示の切り替え

- **グリッド/リスト**: 右上のセレクトボックスで、Finderのアイコン表示とリスト表示を切り替えられます。
- **サイズ変更**: グリッド時のサムネイルサイズを「大・中・小」から選べます。

## 📁 フォルダ構成と除外ルール

- `assets/` : カタログのルート。ここに `index.html` と `list.json` を配置。
- `assets/move/` : 動画素材などを自由に整理。
- **除外設定 (`IGNORE_DIRS`)** :
  以下の名前を含むフォルダは、スキャン対象から**完全に除外**されます。
  > `backup`, `old`, `temp`
  >

## 📝 技術メモ

- **URLコピー**: ボタンを押すと `raw.githubusercontent.com` の直リンクをクリップボードにコピーします。
- **プレビュー**: 動画はマウスホバーで再生、画像はクリックでフルサイズ（別タブ）表示されます。
