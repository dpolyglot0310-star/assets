# 📦 Asset Catalog + Pixel Indexer

プライベートグループ（最大5名）向けの素材管理・ドット変換ツールです。
GitHub Pages 上で動作し、サーバー不要でブラウザだけで使えます。

### 🚀 開く

**[👉 カタログを表示する](https://dpolyglot0310-star.github.io/assets/)**

タブ直リンク:
- `?tab=assets` — 素材カタログ
- `?tab=links` — 外部リンク集
- `?tab=pixel` — Pixel Indexer

## 🗂 タブ構成

### Assets & Docs タブ
素材ファイルの一覧・プレビュー・URL管理。

- 画像・動画・Markdownファイルをフォルダ別に表示
- 1MB超のファイルは初期非表示（「表示」ボタンで個別ロード）
- 「全表示」ボタンで一括ロード
- グリッド / リスト 表示切り替え、サムネイルサイズ変更
- URLコピーボタン（`raw.githubusercontent.com` 直リンク）
- 動画はホバーで再生、画像はクリックで別タブ表示
- `.` 始まりフォルダは「🙈 HIDDEN」セクションに隔離
- **「→ Pixel」ボタン**で画像を Pixel Indexer に直接送信

### External Links タブ
`links.csv` で管理する外部リンク・プロンプト集。

### Pixel Indexer タブ
画像をピクセルアートに変換するツール。詳細は下記。

## 🎨 Pixel Indexer 使い方

### 画像の読み込み
- ファイル選択から直接読み込む
- Assets タブの画像カードにある「→ Pixel」ボタンで送信

### 変換設定

| 設定 | 内容 |
|------|------|
| Pixel Size | マス目のサイズ（2〜60px） |
| Color Steps | 減色ステップ数（1〜128） |
| Dithering ON | Floyd-Steinberg 誤差拡散ディザリング |
| Raw（色変換なし） | 減色・ディザをスキップして元色のままグリッド化 |
| BG | キャンバス背景色 |
| Grid | 網目表示のオン/オフ・色・太さ |
| Zoom | 表示倍率（1〜5倍） |
| Max Size | 読み込み時の最大サイズ（長辺をこのpxに縮小、0=無制限） |

### クロップ（範囲切り出し）
1. 画像読み込み後に **✂ 範囲選択** ボタンをクリック
2. キャンバス上をドラッグして使いたい範囲を指定
3. **✔ ここで切り出す** で確定 → その範囲だけをドット変換
4. **↩ 全体に戻す** で元画像全体に戻す

> 切り出し後は処理対象ピクセルが減るため、大きい画像の一部だけ使う場合は処理が軽くなります。

### パレット操作
- 変換後の使用色が輝度順に一覧表示される
- 色ボックスをクリック → その色のマスをハイライト＋インデックス番号表示
- カラーピッカーで個別に色置換
- **チェックボックスで複数選択** → 一括変更バーが出現
  - カラーピッカーで任意の色に一括変換
  - ドロップダウンで現在のパレット内の色を選んで変換
  - リセットで選択色の変換を一括解除

### 保存
- **💾 保存** ボタンで現在のキャンバスを `pixel.png` としてダウンロード

## 🛠 list.json の更新手順

新しい素材を追加したときは以下の手順でインデックスを更新してください。

1. `assets/` フォルダにファイルを配置して GitHub へ `push`
2. カタログページを開き **「APIスキャン」** をクリック
3. スキャン完了後 **「JSON保存」** で `list.json` をダウンロード
4. `assets/list.json` を上書きして GitHub へ `push`

## 📁 フォルダ構成

```
assets/
├── index.html      # メインページ（全タブ統合）
├── list.json       # 素材インデックス（APIスキャンで生成）
├── links.csv       # 外部リンクデータ
├── DEVPLAN.md      # 今後の開発計画
└── README.md       # このファイル
```

**除外設定 (`IGNORE_DIRS`)**: 以下の名前を含むフォルダはスキャン対象から除外されます。
> `backup`, `old`, `temp`
